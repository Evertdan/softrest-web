A striking, minimal, and secure login screen for a restaurant POS system. ALL UI LABELS MUST BE IN SPANISH.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Responsive
- Theme: Light, sophisticated, focus-driven
- Background: Split screen or subtle warm Terracotta-tinted neutral background
- Surface: Pure Surface (#FFFFFF) for the login card
- Primary Accent: Terracotta (#C05621)
- Text Primary: Charcoal Ink (#18181B)
- Inputs: Label above, solid focus rings. No floating labels.

**Page Structure:**
1. **Layout:** Split-screen design (left side: beautiful high-quality Mexican restaurant ambiance image, right side: clean white login panel) OR a single centered card.
2. **Branding:** "SoftRest" logo, prominent at the top of the form.
3. **Welcome Text:** "Bienvenido de nuevo" (Welcome back), "Ingresa a tu cuenta para continuar" (Sign in to continue).
4. **Form Fields:** 
   - Correo electrónico (Email) with mail icon
   - Contraseña (Password) with eye icon for toggle visibility
5. **Form Controls:** "Recordarme" (Remember me) checkbox, "Olvidé mi contraseña" (Forgot password) subtle link.
6. **Submit:** Full-width primary button "Iniciar Sesión" (Sign In) in Terracotta.
7. **Error State (Optional inclusion):** Show a subtle inline red error message below the password field.
