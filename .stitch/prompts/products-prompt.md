A visually organized product catalog and menu management screen for a Mexican restaurant POS. ALL UI LABELS MUST BE IN SPANISH.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first (responsive)
- Theme: Light, visual, gallery-airy
- Background: Canvas Sand (#FAFAF9)
- Primary Accent: Terracotta (#C05621)
- Text Primary: Charcoal Ink (#18181B)
- Product Cards: Subtle borders, hover elevation, generous border radius (1rem)
- Images: Rounded top corners, clear food photography placeholders

**Page Structure:**
1. **Sidebar Navigation:** "Productos" is active.
2. **Header:** Title "Menú y Productos", with a primary "Agregar Producto" button.
3. **Control Bar:** 
   - Search input ("Buscar producto...")
   - Category Tabs (e.g., Tacos, Bebidas, Entradas, Postres). Active tab has a Terracotta bottom border or pill background.
4. **Product Grid:** A responsive CSS grid of product cards.
5. **Product Card Layout:** 
   - High-quality placeholder image (top)
   - Title (e.g., "Tacos al Pastor")
   - Category tag
   - Price in Mono font (e.g., "$120.00")
   - Stock indicator badge (e.g., "Disponible", "Agotado" in Rose #E11D48)
   - Hover state reveals quick edit/delete actions
