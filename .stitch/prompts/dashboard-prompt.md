A sophisticated, modern POS dashboard for a Mexican restaurant, blending warm Terracotta accents with clean utilitarian layouts. ALL UI LABELS MUST BE IN SPANISH.

**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first (responsive)
- Theme: Light, professional, airy but functional
- Background: Canvas Sand (#FAFAF9)
- Surface: Pure Surface (#FFFFFF)
- Primary Accent: Terracotta (#C05621) for primary actions and active states
- Text Primary: Charcoal Ink (#18181B)
- Text Secondary: Muted Steel (#71717A)
- Mono Font: JetBrains Mono for all numbers, prices, and metrics
- Display Font: Satoshi
- Cards: Generously rounded (1rem) with diffused whisper shadow
- Buttons: Subtly rounded (8px), tactile active states
- Anti-patterns: No Inter font, no emojis, no fake 99.99% data.

**Page Structure:**
1. **Sidebar Navigation (Left):** Logo "SoftRest", navigation links (Inicio, Órdenes, Productos, Cocina, Configuración) with clean icons. "Inicio" is active (Terracotta highlight). Bottom user profile.
2. **Top Header:** Greeting ("Hola, [Nombre]"), current date/time, and a notification bell icon.
3. **KPI Metrics Grid:** 4 asymmetric cards showing: Ventas del día (Daily sales), Órdenes activas (Active orders), Mesas ocupadas (Tables occupied), Ingresos (Revenue). Use Mono font for numbers. No 3-column equal layouts.
4. **Main Chart Section:** A clean, minimal line chart visualizing "Ventas por hora" (Sales over time).
5. **Recent Activity Panel (Right/Bottom):** "Órdenes Recientes" list showing recent order IDs, table numbers, total amounts, and status badges.
6. **Quick Actions:** A section with prominent buttons for frequent actions: "Nueva Orden", "Corte de Caja" (Register Cut).
